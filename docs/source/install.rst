安装&更新
======================================

在这一章节中,我们介绍一下如何安装pithy-test.

pip安装
--------------------------------------

你可以使用pip安装或更新，方式如下::

    >>> pip install pithy-test
    >>> pip install -U pithy-test

删除
--------------------------------------

::

    >>> pip uninstall pithy-test
    
获取源码
--------------------------------------

::

    >>> git clone https://github.com/yuyu1987/pithy-test.git
